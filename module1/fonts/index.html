How to setup downloaded fonts

  1. Download the fonts you want:
    • Super Mario Fonts
  2. Extract the .woff and font.css files. Then drag the .woff file to the css folder of your project.
  3. Add the custom font css rules from the font.css file to your styles.css file:
    @font-face { font-family: 'SuperMarioBros'; src: local('SuperMarioBros'), url('./SuperMarioBros.woff') format('woff'); } /* use this class to attach this font to any element i.e.

    Text with this font applied

    */ .fontsforweb_fontid_181 { font-family: 'SuperMarioBros' !important; } /* Font downloaded from FontsForWeb.com */
  4. Finally add the class to the element you want to apply the custom fonts to:
    h1 class="fontsforweb_fontid_181"> How to setup downloaded fonts /h1>
